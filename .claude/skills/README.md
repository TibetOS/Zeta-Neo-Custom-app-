# Android skills (vendored)

These skills are vendored from Google's official
[android/skills](https://github.com/android/skills) repository — AI-optimized
instructions that ground coding agents in Android best practices from
developer.android.com. They follow the open
[agent skills](https://agentskills.io/home) standard (`SKILL.md` per skill),
which Claude Code discovers automatically from `.claude/skills/`.

License: Apache 2.0 (see `ANDROID-SKILLS-LICENSE.txt`).

To update, re-copy the skill directories from a fresh clone of
`https://github.com/android/skills`.

Skills most relevant to this project (Compose launcher/dashboard app):
`adaptive`, `styles`, `navigation-3`, `edge-to-edge`, `testing-setup`,
`r8-analyzer`, `android-intent-security`, `agp-9-upgrade`.
